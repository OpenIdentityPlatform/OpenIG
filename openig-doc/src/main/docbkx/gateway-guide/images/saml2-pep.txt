title SAML 2.0 Policy Enforcement Point
autonumber

participant "Browser" as Browser
participant "Access Manager" as AM
participant "${projectName}" as Gateway
participant "Payroll App" as Portal

Browser->Gateway: Request to Payroll App

Gateway->Portal: Pass request through

Portal->Gateway: No session, redirect to login

Gateway->Browser: Intercept login request &...

Browser->AM: ...send SAML 2.0 AuthN request

AM<->Browser: Authenticate user

AM->Browser: SAML 2.0 POST AuthN...

Browser->Gateway: ...auto-submitting form

Gateway->Portal: POST Payroll app login form

Portal->Gateway: Redirect to Payroll App

Gateway->Browser: Pass through redirect from Payroll App
