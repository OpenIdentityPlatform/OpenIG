title Login With Credentials From a File
autonumber

participant "Browser" as Browser
participant "${projectName}" as Gateway
participant "User File" as File
participant "Application" as App

Browser->Gateway: Send HTTP GET request to http://www.example.com:8080/file

Gateway->File: Look up the record containing credentials

Gateway->Gateway: FileAttributesFilter stores credentials in exchange

Gateway->Gateway: StaticRequestFilter retrieves credentials from exchange

Gateway->App: Replace request with HTTP POST of login form

App->Gateway: Validate login & send response

Gateway->Browser: Response from http://www.example.com:8080
